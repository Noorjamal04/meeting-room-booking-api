## Prompt 1

Create a simple Express API using TypeScript for booking meeting rooms.

Requirements:
- In-memory storage
- Create a booking
- Cancel a booking
- List bookings for a room

Business rules:
- Bookings must not overlap
- Bookings cannot be in the past
- Start time must be before end time

Keep it simple and runnable.
